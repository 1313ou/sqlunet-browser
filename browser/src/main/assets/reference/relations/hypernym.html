Hypernym
Superordinate (i.e. more generic). The generic term used to designate a whole class of specific instances.
X is a hypernym of Y iff
→ Y is a (kind of) X (nouns)
→ to Y is to X in some particular manner (verbs)
oaktreean oak is a kind of tree
treeplanta tree is a kind of plant
housedwellinga house is a kind of dwelling
