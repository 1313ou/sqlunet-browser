Direct access to entities is made possible by referencing them by IDs with the following syntax
Pointers
Specific items can be accessed directly by pointing at them in the query, with the following syntax (xxx stands for the item's ID):
pointer item
#wsxxx WordNet synset
#wkwww%k:kk:kk:: WordNet sensekey
#vcxxx VerbNet class
#prxxx PropBank roleset
#ffxxx FrameNet frame
#flxxx FrameNet lexunit
#fsxxx FrameNet sentence
#faxxx FrameNet annotation set
#fpxxx FrameNet pattern
#fvxxx FrameNet valence unit
#mrxxx PredicateMatrix role
